GitHub Sync — Personal Access Token

Enter your GitHub Personal Access Token so that every saved idea is committed directly to github.com/AlexSheff/1149 as data/ideas.json. Your ideas will be version-controlled, synced across devices, and never lost.

How to create the token: 1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Note: 1149 ideas · Expiration: No expiration
4. Scopes: check only repo (full control of private repositories)
5. Click Generate token → copy the ghp_... value here
Token is stored only in your browser's localStorage. It is never sent anywhere except api.github.com.
