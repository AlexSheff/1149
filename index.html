GitHub Sync
💡 Ideas are saved to github.com/AlexSheff/1149 as both data/ideas.json and individual WHITE_PAPER.md files. File attachments are committed to the idea's layer folder. Without a token, data is saved locally only.
How to create a Personal Access Token ▾
1. github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token → Note: "1149 dashboard" → Expiration: No expiration
3. Scopes: check repo only
4. Copy the ghp_… token here and click Save.
Token stored in browser localStorage only — sent only to api.github.com.
