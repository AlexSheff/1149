# Global Thinking Framework

> *Principles of borderless cognition — planetary scale, local precision, simultaneous.*

---

## 1. The Core Principle

**Think globally, implement locally.**

Every system I build must be:
- **Architecturally global** — anyone can fork it, deploy it, adapt it to their context
- **Physically anchored** — specifically implemented in one real place (Dumaguete)

Abstraction without implementation is noise. Implementation without architecture is a dead end.

---

## 2. Dual-Mode Cognition

The productive mode is operating at both ends of the abstraction stack simultaneously:

| Mode | Examples |
|------|---------|
| **Extreme abstraction** | Cosmology, consciousness theory, information as reality |
| **Extreme concreteness** | Which barangay, which person, which line of code |

The tension between these modes is where ideas crystallize into systems. Staying only at one end produces either daydreaming or bureaucracy.

---

## 3. Non-Linear, Timeless Mapping

A project plan has deadlines. A **mental map** has topology.

The City Quest Miro board (`o9J_koZNAuk=`) is not a roadmap — it is a topological map of a territory being explored. Ideas connect when they are ready to connect. This repository operates the same way.

Principles:
- Ideas do not expire — they wait
- Connections emerge through accumulation, not scheduling
- The map is always incomplete — that is a feature, not a bug

---

## 4. Language as Research Object

Every language encodes a different model of reality:
- **Russian** — encodes depth, ambiguity, compression of meaning
- **English** — encodes precision, agency, explicit causation
- **Cebuano** — encodes community, place, relational identity

Working across multiple languages is not a convenience — it is an epistemological practice. Each language reveals blind spots in the others.

---

## 5. The Local Anchor

Dumaguete is not a temporary base. It is a deliberate anchor.

The choice of a specific physical place is the act that makes global thinking real rather than abstract. Without Dumaguete, the Manifesto is philosophy. With Dumaguete, it is practice.

What Dumaguete provides:
- Manageable scale (~130,000 people) for real civic experiments
- Active civil society for genuine community partnership
- Geographic specificity that forces concrete implementation
