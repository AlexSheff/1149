# Fair BEAT — White Paper

> *Structural redistribution mechanics embedded in every transaction — solidarity by design, not by donation.*

---

## Transaction Flow
```
Any City Quest payment (e.g. 1000 PHP)
  → Smart contract intercepts
  → 930 PHP → recipient
  → 70 PHP (7%) → Community Pool
    → 40% distributed immediately (trust-weighted contributors)
    → 30% → Community Treasury (governance-controlled)
    → 20% → Protocol development fund
    → 10% → Burn / deflationary sink
```

### Distribution Formula
```
share_i = trust_score_i / sum(all active trust scores)
```

### Key Principle
The mechanism enforces redistribution. Human virtue is not required.

---

## Solidarity Mechanism

Redistribution happens in the constructor, not in a donation form.
