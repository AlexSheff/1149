## How to use this registry

1. **New idea?** Add it to **Concept / Seed** immediately — one line, no docs needed
2. **Ready to develop?** Move to the right status table, create the folder, copy the template
3. **Searching?** `Cmd+F` on tags — every idea has 3–6 searchable tags
4. **Checking dependencies?** See `CONNECTIONS.md`
